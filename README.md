# Sales Magician - Demo

A **design-only**, click-through prototype of the Mateos.ai _Sales Magician_,
built to align product + leadership on the UX/UI ahead of **0.0.0 (alpha)**.

> Nothing here is real. No backend, no auth, no data - every button just
> navigates. The point is look, feel and flow.

## Stack

Vite · React · TypeScript · Tailwind CSS · shadcn/ui (new-york) - the same
UI library and Mateos brand theme as the production `magician` app:
amber `#FF8C42` (primary) · indigo `#5B53E8` (secondary) · teal `#17BEBB` (accent).

## Run

```bash
npm install
npm run dev        # http://localhost:5173
```
